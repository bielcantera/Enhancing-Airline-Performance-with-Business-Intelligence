# Data (not included)

Raw datasets are not included in this repository due to size and licensing.

To run the project locally, place the required files in this `data/` folder.
Examples of expected files (names may vary depending on your download source):

- ONTIME_REPORTING_01.csv ... ONTIME_REPORTING_12.csv
- train_val.csv (generated by Module 3)
- weather.csv (or equivalent)
- aircraft inventory / carriers / passengers / employees datasets
- satisfaction_P5.csv (Module 5)

Update file paths inside scripts/notebooks if your filenames differ.
